# ellenmartin11.github.io

### About Me!
Hi, I'm Ellen! I'm currently working as a Postgraduate Research Assistant at the Yale Rutledge Lab. I help to collect data for a longitudinal R01 aiming to understand decision-making and happiness using smartphone app data. I am also working on a project that uses free text-responses to understand language use in depression. 

### Education
I completed my BSc in Psychology at UCL in 2021, graduating with an Upper First Class Degree, and was on the Dean's List. My third year dissertation was nominated for the Undergraduate Project Prize Award. You can read more about this project [here](https://github.com/ellenmartin11/lone-GenSEM-MR), or you can read the [Psychiatry Research 2023 publication](https://www.sciencedirect.com/science/article/pii/S0165178123001695)! I then completed my MRes in Developmental Neuroscience and Psychopathology, which was a joint Yale-UCL programme. My Master's thesis focused on [the associations between prenatal cocaine exposure, substance use and young adult intimate partner violence](https://github.com/ellenmartin11/prenatal-cocaine-exposure-and-dating-violence/tree/main).
